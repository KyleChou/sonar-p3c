循环体内，字符串的联接方式，使用StringBuilder的append方法进行扩展.

Examples:

    Negative example:
    String result;
    for (String string : tagNameList) {
        result = result + string;
    }

    Positive example:
    StringBuilder stringBuilder = new StringBuilder();
    for (String string : tagNameList) {
        stringBuilder.append(string);
    }
    String result = stringBuilder.toString();
